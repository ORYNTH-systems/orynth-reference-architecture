## Purpose

Let

O(t)

represent the constitutional orientation state at constitutional observation time t.

## Orientation Mapping

Define

Ω : O → F

where Ω maps an orientation state to the constitutionally admissible framework set available at that stage of orientation.

## Dependency Constraint

Let

D(fi,fj)

represent constitutional dependency.

## Provenance Preservation

Let

P(fi)

represent the provenance function of framework fi.

For every orientation state,

P(fi,t₁) = P(fi,t₂)

## Semantic Preservation

Let

S(fi)

represent the constitutional semantic identity of framework fi.

For every orientation transition,

S(fi,t₁) = S(fi,t₂)

Orientation preserves meaning.

It does not redefine meaning.

---

## Orientation Reconstruction

Let

R(O)

represent reconstruction of an orientation state.

Constitutional equivalence exists when

R(O) = O

using identical admitted constitutional evidence.

- dependency ordering;

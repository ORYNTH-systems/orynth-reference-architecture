# Morning Star Governance

## Purpose

The governance layer preserves the constitutional continuity of Morning Star after completion of the Canonical Model.

It governs constitutional maintenance rather than constitutional meaning.

The Canonical Model defines the constitutional architecture.

The Governance layer preserves that architecture across future revisions.

## Constitutional Rule

No governance document may alter canonical meaning.

Canonical meaning is established only within:

constitution/CANONICAL_MODEL.md

Governance documents preserve, review, version, and steward that meaning.

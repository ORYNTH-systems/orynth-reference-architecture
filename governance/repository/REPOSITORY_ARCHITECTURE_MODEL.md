## Purpose

Define the repository architecture governing ORYNTH systems, foundations, applications, and successor architectures.

---

## Core Principle

A repository represents an independent architectural identity, not merely a storage location.

Repository boundaries follow:

- purpose;
- lifecycle;
- governance;
- ownership;
- release identity;
- citation value.

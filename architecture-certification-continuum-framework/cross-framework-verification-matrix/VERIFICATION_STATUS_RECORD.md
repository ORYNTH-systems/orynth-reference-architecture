# Verification Status Record

Version:

v1.0.0

---

## Purpose

Define the record structure for cross-framework verification activities.

---

## Record Fields

Includes:

- verification identifier;
- evaluated framework;
- related framework;
- verification criteria;
- evidence references;
- result;
- certification relevance.

---

## Verification Lifecycle


VERIFICATION INITIATED

↓

FRAMEWORK RELATIONSHIP IDENTIFIED

↓

CRITERIA EVALUATED

↓

EVIDENCE REVIEWED

↓

RESULT RECORDED

↓

STATUS PRESERVED


---

## Verification States

Possible states:

- IDENTIFIED;
- MAPPED;
- REVIEWING;
- VERIFIED;
- CERTIFIED;
- HISTORICAL.

---

## Principle

Verification Status Records preserve the evidence chain supporting architecture certification.

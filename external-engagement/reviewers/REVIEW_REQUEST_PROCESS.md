# Review Request Process

Version:

v1.0.0

---

## Purpose

Define the process for requesting external architecture review.

---

## Review Request Includes

- architecture overview;
- review scope;
- supporting documentation;
- expected feedback areas.

---

## Review Stages


PREPARE MATERIALS

↓

CONTACT REVIEWER

↓

PROVIDE ACCESS

↓

RECEIVE REVIEW

↓

DOCUMENT RESPONSE


---

## Principle

Review requests should provide sufficient context for meaningful evaluation.

# Decision Analysis Framework

Version:

v1.0.0

---

## Purpose

Define analytical methods used to evaluate decisions and possible outcomes.

---

## Analysis Factors

Includes:

- objectives;
- constraints;
- available evidence;
- alternatives;
- risks;
- expected outcomes.

---

## Decision Analysis Lifecycle


DEFINE PROBLEM

↓

IDENTIFY OPTIONS

↓

COMPARE CONDITIONS

↓

EVALUATE CONSEQUENCES

↓

SELECT PATH

↓

DOCUMENT RESULT


---

## Principle

Decision analysis creates structured understanding before action is taken.

# Failure Impact Model

Version:

v1.0.0

---

## Purpose

Define the model used to evaluate architectural failure impact.

---

## Impact Principle

Failure impact is determined by the affected architectural properties and the ability to preserve continuity.

---

## Impact Assessment Model


FAILURE CONDITION

↓

AFFECTED COMPONENT

↓

IMPACT CATEGORY

↓

SEVERITY EVALUATION

↓

RECOVERY REQUIREMENT

↓

RESILIENCE DETERMINATION


---

## Impact Categories

### Identity Impact

Effect on architecture definition and purpose.

### Functional Impact

Effect on intended capability.

### Governance Impact

Effect on controlled decision processes.

### Evidence Impact

Effect on traceability and verification.

### Continuity Impact

Effect on connection between architecture states.

---

## Severity Levels

Possible states:

- LOW;
- MODERATE;
- HIGH;
- CRITICAL.

---

## Principle

The Failure Impact Model provides structured understanding of how disruption affects architectural resilience.

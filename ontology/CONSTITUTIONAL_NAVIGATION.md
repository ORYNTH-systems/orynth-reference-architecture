Status: ACTIVE

Version: 0.1.0

## Purpose

This document defines the constitutional navigation model used by Morning Star to orient an initiate through the ORYNTH ecosystem.

Navigation governs learning progression.

Navigation does not alter constitutional authority.

---

## Navigation Principles

Constitutional navigation shall:

- preserve framework independence;
- preserve semantic boundaries;
- preserve provenance;
- preserve reconstructability;
- preserve constitutional ordering.

An initiate shall be able to determine:

- current constitutional location;
- previous stage;
